# 🏥 Doctors Plaza

**Find Trusted Doctors Near You by Location and Specialty**

Doctors Plaza is a modern web platform that helps users find trusted doctors based on their location and specialty. It also allows users to rate doctors and share their experiences to guide others in making better healthcare decisions.

## 🚀 Features

### 📍 Location-Based Search
Find doctors near you using your current location or by entering your county and sub-county.

### 🩺 Doctor Categories
Browse doctors by specialties including:
- Cardiology
- Dermatology
- Neurology
- Pediatrics
- Orthopedics
- Gastroenterology
- Endocrinology
- Radiology
- Anesthesiology
- General Practice

### ⭐ Rating System
- Rate doctors from 1-5 stars
- View average ratings and feedback from other users
- Help others make informed healthcare decisions

### 💬 User Reviews
- Share your experience with doctors
- Read authentic reviews from other patients
- Make better healthcare choices based on real experiences

### 🎯 Simple & Clean UI
- Modern, responsive design built with Tailwind CSS
- Mobile-friendly interface
- Fast and intuitive user experience

## 🛠️ Technologies Used

- **HTML5** - Semantic markup and structure
- **Tailwind CSS** - Utility-first CSS framework for styling
- **JavaScript (ES6+)** - Interactive functionality and DOM manipulation
- **Geolocation API** - Location-based doctor search
- **Font Awesome** - Icons and visual elements

## 📦 How to Use

1. **Open the website** in your web browser
2. **Find doctors near you:**
   - Click "Use My Location" to automatically detect your area
   - Or manually enter your county and sub-county
3. **Filter by specialty** using the category buttons
4. **View doctor profiles** with ratings and reviews
5. **Book appointments** directly through the platform
6. **Share your experience** by rating and reviewing doctors
